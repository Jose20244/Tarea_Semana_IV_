El programa solicita los datos del usuario que está cargando los productos.
Cada producto se guarda en un nodo que se crea dinamicamente en memoria usando New.
Los nodos se enlazan uno con otro formando una lista, mantendiendo un puntero al primer nodo de la lista.
El programa recorre la lista utilizando un puntero aux para mostrar datos de cada producto y calcular el total de los precios.
Y por ultimo se libera correctamente la memoria dinámica de cada nodo utilizando delete, evitando fugas de memoria.
